1. Используемые версии: Python 3.7, MySQL 5.7.

2. Управляющий модуль - файл main.py.

3. В задании по второму плагину про SQL-запрос неясно, откуда должны поступать значения для добавления в БД.
Для проверки дублирования номеров  вижу два варианта:

а) В самой базе данных поля imsi и iccid сделать уникальными. Либо сразу при создании, либо использовать код:
ALTER TABLE imsi
ADD UNIQUE (imsi);
ALTER TABLE iccid
ADD UNIQUE (iccid).

b) Допустим, что нет доступа для изменения структуры таблиц. Для этого случая я написал код в самом файле второго плагина. Там применяется временная таблица, в которую заносятся введённые номера из файла.
